# 実務イメージ型 ポートフォリオ
【概要】  
実務を想定した制作物を業務外の時間で作成しました。  
よって、本制作物は過去に在籍していた会社や現場で直接的に使用していたファイルではありません。  
本リポジトリはフォルダ単位で制作物を格納しています。  
制作物の主な内容は実行プログラム、設計書、操作説明書、試験項目になります。  

---
## ZIPファイルのダウンロード方法
1. [リポジトリURL](https://github.com/Test-Information/portfolio)にアクセス
2. 緑色の「**<>CODE▼**」ボタン押下
3. 一番下の「**Download ZIP**」押下

---
## ZIPファイルのダウンロード以外の方法
会社のセキュリティルールなどで外部サイトのプログラムをダウンロードできない場合を想定し、  
Officeファイルに関してはPDF版を用意いたしました。   
一部抜粋になりますが、以下のリンクをクリックすることでブラウザ上で閲覧可能です。   
※すべて確認する場合はダウンロードが必要です。   
- [ソースコード](https://github.com/Test-Information/portfolio/blob/main/03_StartUpFiles/src/StartUpFiles.cpp)
- [設計書](https://github.com/Test-Information/portfolio/blob/main/01_FileNameSearch/docs/pdf/01_Program_Design_v1.0.0.pdf)
- [試験項目および証跡](https://github.com/Test-Information/portfolio/blob/main/01_FileNameSearch/docs/pdf/01_TestCase_v1.0.0.pdf)
- [操作説明書](https://github.com/Test-Information/portfolio/blob/main/01_FileNameSearch/docs/pdf/01_How_to_use_v1.0.0.pdf)

# 成果物イメージ型 ポートフォリオ
実務を想定した制作物を掲載しています。  
リポジトリ内のフォルダ単位で制作物を格納しています。  
制作物の主な内容は実行プログラム、設計書、操作説明書、試験項目になります。  

---
## ZIPファイルのダウンロード方法
1. [リポジトリURL](https://github.com/Test-Information/portfolio)にアクセス
2. 緑色の「**<>CODE▼**」ボタン押下
3. 一番下の「**Download ZIP**」押下


## 成果物一覧
| 制作区分 | 制作物名                    | 説明                                               | 備考      |
| -------- | --------------------------- | -------------------------------------------------- |---------- |
| 01       | ファイル名検索バッチ        | 指定のディレクトリ配下からファイルを検索する       |           |
| 02       | OracleDBユーザー選択バッチ  | OracleDBのログインを効率化する                     |           |


## セキュリティに関して
会社独自のセキュリティルールなどで外部サイトのプログラムをダウンロードできない場合などは、  
直接ファイルをクリックしていただくだけでもソースコードを確認することができます。

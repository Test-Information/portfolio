# 成果物イメージ型 ポートフォリオ
実務を想定した制作物を掲載しています。  
リポジトリ内のフォルダ単位で制作物を格納しています。  
制作物の主な内容は実行プログラム、設計書、操作説明書、試験項目になります。  

---
## ZIPファイルのダウンロード方法
1. [リポジトリURL](https://github.com/Test-Information/portfolio)にアクセス
2. 緑色の「**<>CODE▼**」ボタン押下
3. 一番下の「**Download ZIP**」押下

## ZIPファイルのダウンロード以外の方法
会社のセキュリティルールなどで外部サイトのプログラムをダウンロードできない場合を想定し、
Officeファイルに関してはPDF版を用意いたしました。 
運用を効率化するため、現状、PDF版はv1.0.0に限定させていただきます。
よって、ソースコード、設計書、試験項目、試験証跡、操作説明書がダウンロードせずクリックのみで、ご確認していただけます。
